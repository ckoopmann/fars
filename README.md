# Fars Package
This is the *fars* package as part of the Coursera Course *Building R Packages*.

## Functionality

This package implements the following functions:

1. Read in Accident data from a csv file for given years
1. Summarize Accident data into monthly counts for a given year
1. Visualize Accidents on a states map

## Documentation
To find out more about the implementation of these functions and how to access them, read the vignette *Fars Package Introduction*.
